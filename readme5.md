- create function to determine the given expression is correct or not
- expression will be string of open bracket nor close bracket e.g `({[[()]]})`
- your function should determine the expression is it correct or not
- examples:
  - `soal("({[]})")` => true
  - `soal("([][]{})")`=> true
  - `soal("({)(]){[}")` => false
  - `soal("[)()]")` => false
